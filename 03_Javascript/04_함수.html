함수

1. 함수 선언

1) 함수

function 키워드에 함수명을 작성하여 선언하는 방법이다.

2) 익명 함수

함수명을 작성하지 않고 선언하는 방법이다. (이벤트 핸들러 등록 시 주로 사용)

3) 화살표 함수

ES6에 도입된 함수 선언 방법으로 fucntion 키워드 대신 '=>' 를 사용해서 좀 더 간략한 방법으로 함수를 선언할 수 있다.

2. 함수 호출

1) 매개 변수

함수 실행에 필요한 값은 매개변수를 통해 함수에 전달할 수 있다.

2) arguments 객체

초과되는 매개값은 arguments 객체를 통해 접근이 가능하다.

3. 함수 리턴

1) 일반적인 값 리턴

2) 익명 함수 리턴

4. 내장 함수

1) eval() 함수

문자열을 자바스크립트 코드로 변환하여 실행한다.

2)isFinity(), isNaN() 함수

isFinity() : 숫자가 유한한 값인지 확인한다.
isNaN() : 자바스크립트로 표현할 수 없는 숫자(NaN)인지 확인한다.
