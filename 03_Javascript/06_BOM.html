BOM(Browser Object Model

1. window 객체

브라우저 창에 대한 설정을 하는 가장 최상위 객체이다.

1) window.open()

2. 타이머(Timer)

window.setTimeOut()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 단 한 번만 실행한다.

window.setInterval()

일정 시간이 경과된 이후에 매개값으로 전달된 콜백 함수를 반복해서 실행한다.

2. location 객체

브라우저의 주소 표시줄(URL)과 관련된 객체로 주소와 관련된 정보를 가진 객체이다.

1) location.reload()

페이지를 새로 고침하는 메소드이다.

2) location.href

현재 페이지의 URL 주소를 가지고 있는 속성이다.
해당 속성의 값을 변경해서 페이지를 이동할 수 있다.

3) location.assign(), location.replace()

두 메소드 모두 페이지를 이동시키지만 replace()는 assign()과 다르게 history에 기록되지 않는다.
assign은 history가 남아서 뒤로가기가 가능하다.
replace는 history가 남지 않아서 뒤로가기가 불가능하다.

3. history 객체

브라우저에서 이동한 문서의 내역을 관리하는 객체이다.

4. navigator 객체

브라우저에 대한 정보를 가지는 객체이다.

5. screen 객체

사용자의 디스플레이에 대한 정보를 가진 객체이다.
