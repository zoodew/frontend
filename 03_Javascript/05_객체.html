객체

1. 객체 생성

자바스크립트는 다양한 바업으로 객체를 생성할 수 있다.

2. 메소드

프로퍼티의 값이 함수인 경우를 메소드라고 한다.

3. 객체의 속성 추가 및 삭제

프로퍼티와 메소드는 객체 생성 후 동적으로 추가/삭제할 수 있다.

4. 객체 배열

객체와 배열을 사용해서 데이터를 관리할 수 있다.

5. 생성자 함수

new 연산자와 함께 호출되어 객체를 생성하는 함수를 생성자 함수라 한다.

6. 캡슐화

객체 생성 후 변수를 통해 속성에 직접 접근이 가능하다. 직접 접근이 가능하기 때문에 변수를 임의로 바꾸는 것도 가능함 이걸 막기 위해 캡슐화 구현

지역변수와 클로저를 활용하여 캡슐화를 구현할 수 있다.

7. Date 객체

날짜와 시간 정보를 저장하는 객체이다.
