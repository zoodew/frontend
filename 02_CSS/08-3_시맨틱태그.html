시맨틱 태그

웹 접근성 및 웹 문서의 구조화와 관련하여 기존의 div 형태로 나누었던 구조를
별도의 태그 형식으로 정의한 태그들을 말한다.

    [header]    : 머리말
    [nav]       : 주 메뉴
    [section]   : 내용 
    [article]   : 내용
    [aside]     : 사이드 메뉴 
    [footer]    : 웹 문서의 정보, 저작권 등을 작성하는 영역 


    - id나 class를 사람마다 다 다르게 작성하기 때문에 사회적 약속을 해준 것
    - 태그만 보고도 이게 어떤 기능으로 작용하는지 쉽게 파악할 수 있다.
